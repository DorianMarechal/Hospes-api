Always read TODO.md at the root of the project before starting any work.

**Why:** The TODO.md contains the real roadmap with phases and checked/unchecked items. Without reading it first, work may target the wrong phase or duplicate existing work.
**How to apply:** First action of every session: read TODO.md, then align work with the next unchecked phase/items.
